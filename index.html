GIT BASH $ git config --global user.name "Your name" $ git config --global user.email you@name.com verificar nombre y correo $ git config --list Iniciar un repositorio $ git init * $ ls -la (ver los elementos ocultos) $ git status $ git add Autobiografía.txt $ git add 'Anotaciones Git.txt' * '' comillas si el nombre tiene espacios $ git status $ git commit -m "Esta es la versión inicial" Cambio algo en el archivo de texto u otro $ git status (verificar si se modificó un archivo) $ git add 'Anotaciones Git.txt' (subir el archivo modificado) $ git commit -m "Agregando notas de Git" Versiones $ git log --oneline Chequear una versión anterior $ git checkout 92bf9a3 Volver a la versión actual $ git checkout master Detalles de cada commit y la version $ git log Detalles de cambios de un commit a otro $ git show Agregar todos los archivos modificados $ git add . $ git commit -m "Ejemplo agregando modificaciones de varios archivos" Volver a un commit anterior para siempre(elimina los cambios después de ese commit) $ git reset d4ce2ed(código del commit) --hard RAMAS Ver ramas $ git branch Crear ramas $ git branch biografiamadre Pasar a una rama $ git checkout biografiamadre *merge: unir el head de la rama master con el head de la rama biografiamadre Unir ramas $ git checkout master $ git merge biografiamadre *$ git commit -am "nuevo cambio" : comando para invocar dos comandos git add y git commmit -m en uno solo; pero el archivo ya debió ser agregado por un git add previamente. *conflicto al realizar un merge: borrar todo lo innecesario y dejar lo que vale, luego hacer un commit(git commit -am ...)
